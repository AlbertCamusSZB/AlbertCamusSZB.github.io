元素的居中对齐

要水平居中对齐一个元素，可以使用margin: auto;,设置到元素的宽度将防止它溢出到容器的边缘。元素通过制定宽度，并将两边的空白外距平均分配；

如果没有设置 width 属性(或者设置 100%)，居中对齐将不起作用。
